# Go Switch-Case Coding Problems

## Problem 1

Write a Go program that uses a switch statement to categorize a given integer into the following ranges:
- "Low" for numbers between 0-3
- "Medium" for numbers between 4-6
- "High" for numbers between 7-9"


## Problem 2

Modify the given Go code to use a fallthrough statement in one of the cases.
Explain its effect.


## Problem 3

Write a Go function `switchExample(n int)` that takes an integer `n` as input
and prints a specific message based on the value of `n` using a switch-case statement.


## Problem 4

Convert the normal switch-case statement in the given code into a type switch
that handles different types of input values.


## Problem 5

Implement a Go program that uses a switch statement inside a loop to print
5 random numbers along with their classifications (even/odd).
