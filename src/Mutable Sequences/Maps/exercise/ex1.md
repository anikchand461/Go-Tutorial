# Set Simulation Using Map Questions

### **Question 1: Implement Set with Strings**
Modify the code to simulate a set of strings instead of integers. Use a map with string keys and bool values to store unique string elements. Add a few string values to the set and print the size of the set.

#### Input:
- A slice of string values with duplicates.

#### Output:
- The size of the set (number of unique elements) and some sample set values.

---

### **Question 2: Check Element Existence in Set**
Modify the code to check if a particular element exists in the set. Use a map to check for the presence of the number `8` in the set, and print whether it's found or not.

#### Input:
- A number to check in the set (e.g., `8`).

#### Output:
- Print whether `8` is present in the set or not.

---

### **Question 3: Count Unique Elements from a Slice**
Write a function that accepts a slice of integers and returns the number of unique elements in it using the map approach demonstrated in the code. Call this function with a sample slice of integers and print the result.

#### Input:
- A slice of integers with duplicates.

#### Output:
- The number of unique elements in the slice.

---

### **Question 4: Handle Non-existent Keys**
In the current code, when trying to access an element not in the set (e.g., `500`), the program prints an empty result. Modify the code to handle such cases and print a more informative message when a key is not found in the set.

#### Input:
- A non-existent key (e.g., `500`).

#### Output:
- A message indicating that the element is not present in the set.

---

### **Question 5: Add New Element to Set**
Add a functionality where you prompt the user to add an integer to the set. After adding, print the updated set and its size. Use a map with integer keys and boolean values to simulate the set.

#### Input:
- A user input integer to be added to the set.

#### Output:
- The updated set with the newly added element and its size.
