# Go Looping Constructs Questions

### **Question 1: C-style For Loop**
Write a program that uses a C-style for loop to print the numbers from 1 to 20. 

#### Input:
- The loop should print numbers from 1 to 20.

#### Output:
- Numbers from 1 to 20 printed in a single line.

---

### **Question 2: Condition-Only For Loop**
Write a program that uses a condition-only for loop to print all powers of 3 that are less than 1000.

#### Input:
- Use a condition-only for loop and print powers of 3.

#### Output:
- Powers of 3 less than 1000 (i.e., 1, 3, 9, 27, 81, 243, 729).

---

### **Question 3: Infinite For Loop**
Write a program that uses an infinite for loop to print "Go is awesome!" continuously. Use `break` to exit the loop after printing the message 5 times.

#### Input:
- Infinite loop with "Go is awesome!" printed 5 times.

#### Output:
- "Go is awesome!" printed 5 times.

---

### **Question 4: For-Range Loop Over Slice**
Write a program that uses a for-range loop to iterate over a slice of strings containing names of fruits. Print each fruit name along with its index.

#### Input:
- Slice: `["Apple", "Banana", "Cherry"]`

#### Output:
- Print the index and name of each fruit.

---

### **Question 5: For-Range Loop Over Map**
Write a program that uses a for-range loop to iterate over a map of countries and their capitals. Print both the country name and the capital.

#### Input:
- Map: `{"USA": "Washington D.C.", "India": "New Delhi", "France": "Paris"}`

#### Output:
- Country name and capital printed.

---

### **Question 6: Modify Values in For-Range Loop**
Write a program that demonstrates that modifying the value variable in a for-range loop does not change the underlying value in the slice. Iterate over a slice of integers and try to modify each element's value in the loop.

#### Input:
- Slice: `[1, 2, 3, 4]`

#### Output:
- Original slice should remain unchanged, even though the loop tries to modify the values.
