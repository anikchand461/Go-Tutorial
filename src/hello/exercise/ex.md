# 5 Coding Problems - User Input with If-Else in Go

## Problem 1: Check Positive or Negative
Write a program that asks the user to input a number and then checks if the number is positive or negative using an `if-else` statement.

### Input:
- A single integer.

### Output:
- Print "Positive" if the number is greater than 0, "Negative" if the number is less than 0, or "Zero" if the number is 0.

---

## Problem 2: Check Even or Odd
Write a program that asks the user to input a number and then checks whether the number is even or odd using an `if-else` statement.

### Input:
- A single integer.

### Output:
- Print "Even" if the number is divisible by 2, otherwise print "Odd".

---

## Problem 3: Check Leap Year
Write a program that asks the user to input a year and checks if it is a leap year using an `if-else` statement.

### Input:
- A single integer (year).

### Output:
- Print "Leap Year" if the year is a leap year, otherwise print "Not a Leap Year".

---

## Problem 4: Check Valid Age for Voting
Write a program that asks the user to input their age and checks if they are eligible to vote using an `if-else` statement. In this case, the minimum age for voting is 18.

### Input:
- A single integer (age).

### Output:
- Print "Eligible to Vote" if the age is greater than or equal to 18, otherwise print "Not Eligible to Vote".

---

## Problem 5: Check Valid Password
Write a program that asks the user to input a password and checks if the password meets the requirements: 
- At least 8 characters long,
- Contains both uppercase and lowercase characters,
- Contains at least one numeric digit.

Use an `if-else` statement to print if the password is valid or not.

### Input:
- A single string (password).

### Output:
- Print "Valid Password" if the password meets all the requirements, otherwise print "Invalid Password".
